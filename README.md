A high-speed, 1000mm tall Delta 3D printer featuring a 3030 aluminum frame, HGR15 linear rails, and an integrated enclosure for technical material printing.

## Technical Specifications
Frame: 30x30mm Aluminum Extrusion (1000mm height, 400mm base).

Motion: Delta geometry, 40T pulleys, HGR15 rails (750mm), and Carbon Fiber rods.

Electronics: Undecided based of budget

Power: 24V 300W PSU 

Toolhead: High-flow effector with universal groove mount.

Key Features

High Speed: Optimized lightweight effector and high-torque 24V motion system.


Smart Homing: supports physical endstops

## Future Goals

Precision Leveling: 3-point Auto Bed Leveling support 

Automatic Hotend Changing
